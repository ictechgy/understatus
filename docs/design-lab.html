Design notes
Glyph strategy. The calm ramp uses five minimal Unicode characters: ○ (idle) ▁ (light) ▄ (moderate) ▆ (elevated) ◆ (critical). Each is compact, readable in common terminal fonts, has no semantic color of its own, and escalates visually along a geometric axis — not an emotional one.
  • States 0–3 use only cool blue-greys, increasing brightness without crossing into alert colors. A developer reading code should not register these states as alarms.
  • State 4 is the single hue exception: terracotta #b87848 — warm but desaturated, closer to an aged document than a warning light. The slow opacity pulse (30s runtime default / 5s demo) is perceptible at the edge of attention without demanding it.
  • Chained HUD tail is rendered at 28% opacity with a hairline left border, making it clearly separate yet readable at a glance. understatus's own segment never competes.
  • Separators are mid-dot · at near-invisible gray rather than pipes or slashes, keeping the baseline flat and undemanding.
  • Label/value split: short prefixes (mem disk ctx ⎇) at the dimmest text tier; values one tier brighter. The hierarchy reads in a single glance.
